# Pandas 数据分析训练营 - 产品需求文档

## 1. Product Overview
打造一套完整闭环、可深度学习、实操训练、课后练习、综合测评的专业 Pandas 数据分析实训学习平台，覆盖从零基础入门到实战项目的完整学习路径。

- 解决现有静态网站内容空洞、缺乏交互、实操性弱的问题
- 为数据分析学习者提供系统化、可落地、可测评的完整学习体验

## 2. Core Features

### 2.1 User Roles
| Role | Registration Method | Core Permissions |
|------|---------------------|------------------|
| 学员 | 无需注册 | 浏览课程、完成练习、参与测评、查看进度 |

### 2.2 Feature Module
1. **首页/课程概览页**：课程介绍、10个项目展示、学习路径导航
2. **课程学习页**：章节内容、知识点精讲、代码案例演示
3. **项目任务页**：基础巩固、进阶实操、综合实战三层任务
4. **练习题库页**：单选、多选、代码改错、代码填空、编程实操
5. **测评考核页**：随堂小测、单元测评、阶段考核、结业考评
6. **学习进度页**：进度追踪、完成记录、成绩统计

### 2.3 Page Details
| Page Name | Module Name | Feature description |
|-----------|-------------|---------------------|
| 课程学习页 | 章节内容 | 学习目标、前置知识点、核心理论、代码案例、业务解读、易错点、总结 |
| 项目任务页 | 分层任务 | 10个项目，每个包含基础、进阶、综合三层任务，支持代码提交 |
| 练习题库页 | 交互题库 | 多种题型，隐藏式答案，分步解析，知识点回溯，折叠展开 |
| 测评考核页 | 多维度测评 | 分值标准、合格要求、多维度综合评估 |
| 学习进度页 | 进度追踪 | 完成状态、学习记录、成绩展示 |

## 3. Core Process
用户通过课程概览进入学习，按章节顺序学习课程内容，完成配套练习，参与项目任务和测评考核，系统记录学习进度。

```mermaid
flowchart LR
    A[首页/课程概览] --> B[课程学习]
    B --> C[课后练习]
    C --> D[项目任务]
    D --> E[测评考核]
    E --> F[学习进度]
```

## 4. User Interface Design

### 4.1 Design Style
- **主色调**：深蓝色（#1e3a5f）、科技蓝（#2563eb）
- **辅助色**：青绿色（#10b981）、琥珀黄（#f59e0b）、珊瑚红（#ef4444）
- **按钮风格**：圆角矩形（8px），微阴影，悬停上浮动效
- **字体**：Inter 作为无衬线主体，JetBrains Mono 等宽代码字体
- **布局风格**：卡片式布局，侧边栏导航，清晰的视觉层次
- **图标**：Lucide React 线性图标，简洁专业

### 4.2 Page Design Overview
| Page Name | Module Name | UI Elements |
|-----------|-------------|-------------|
| 首页 | 课程介绍 | Hero 大图、项目卡片网格、课程特色标签 |
| 课程学习页 | 章节内容 | 侧边目录导航、内容卡片、代码高亮块、进度指示器 |
| 练习题库页 | 交互题库 | 题型标签、答题区域、折叠式答案、解析面板 |
| 项目任务页 | 分层任务 | 任务步骤卡片、代码编辑器区域、提交按钮 |
| 测评考核页 | 多维度测评 | 计时器、分值显示、题目卡片、提交确认 |

### 4.3 Responsiveness
- Desktop-first 设计
- 断点：1200px、768px、480px
- 移动端优化：侧边栏折叠为汉堡菜单，卡片垂直堆叠
- 触摸交互优化：增大可点击区域，优化滚动体验

### 4.4 3D Scene Guidance
本项目暂不涉及 3D 场景。
